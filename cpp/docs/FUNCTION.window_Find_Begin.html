Function
windowFindBegin
Initializes the pattern. Sets the finder on the begin position. Gets the first non-repeat range and sets it in the finder. Used together with windowFindBegin and windowFindEnd.
windowFindBegin(finder, pattern, errorRate)
Include Headers
seqan/index.h
Parameters
finder
A SWIFT finder.
pattern
A SWIFT pattern.
errorRate
Error rate that is allowed between reads and reference. Should be the same in as in windowFindNext.
Types: double
